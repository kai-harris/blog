Mac?  Buy an external hard drive, plug it in, use Time Machine.

PC?  Buy an external hard drive that comes with backup software, plug it in, use the backup software.

Linux? You're smart enough to figure it out yourself and it probably involves rsync.

Cloud solution?  Use any number of different providers.

No excuses.  Get it done.  Or, you might find yourself doing this.
